## Useful Shortcuts
	- `Enter` - Enter edit mode
	- `Esc` - Leave edit mode
	- `CMD-K` - Search or create page
	- `g s` - Show shortcuts
	- `g j` - Show journals
	- `g n` - Next journal
	- `g p` - Previous journal
	- `g a` - All pages
	- `g f` - Flashcards
	- `g g` - Graph view
	- `CMD-SHIFT-P` - Command palette
	- `CMD-SHIFT-J` - Open journal in sidebar
	- `t w` - Make screen wider
	- `t l` - Toggle left sidebar
	- `t r` - Toggle right sidebar
